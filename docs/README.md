# 📚 ドキュメント

このディレクトリには、クリーンアーキテクチャとDDDの学習に関するドキュメントが含まれています。

## 📖 メインドキュメント

### 🎯 [完全学習ガイド](COMPLETE_LEARNING_GUIDE.md)
- **対象**: 初心者〜中級者
- **内容**: クリーンアーキテクチャとDDDの基本から実装まで
- **特徴**: 中学生でも理解できるよう丁寧に解説

### 🏗️ [アーキテクチャガイド](ARCHITECTURE_GUIDE.md)
- **対象**: 中級者〜上級者
- **内容**: アーキテクチャの詳細と実装パターン
- **特徴**: より技術的な詳細を含む

## 📁 学習用サンプル

### 🧪 [learning-examples/](learning-examples/)
- **内容**: 実行可能なPHPサンプルファイル
- **用途**: 概念の理解と動作確認
- **特徴**: 段階的に学習できるよう構成

## 📋 学習の進め方

### 🚀 初心者向け

1. **[完全学習ガイド](COMPLETE_LEARNING_GUIDE.md)** を読む
2. **[learning-examples/](learning-examples/)** のサンプルを実行
3. 実際のアプリケーション（`http://localhost:8080`）を操作

### 🎯 中級者向け

1. **[アーキテクチャガイド](ARCHITECTURE_GUIDE.md)** を読む
2. 実際のコードを読んで理解を深める
3. テストを実行して動作を確認

### 🏆 上級者向け

1. コードを改良・拡張してみる
2. 新しい機能を追加してみる
3. 他のドメイン（例：ユーザー管理）を実装してみる

## 🎓 学習目標

このドキュメントを通じて以下を習得できます：

- ✅ クリーンアーキテクチャの基本概念
- ✅ DDDの重要な要素（エンティティ、値オブジェクト、リポジトリ）
- ✅ 依存性逆転の原則
- ✅ テスト駆動開発の基礎
- ✅ 保守性の高いコードの書き方
